🎯 Consejos para selección de variables:

  • • Selecciona variables que tengan relación lógica con la variable objetivo
  • • Evita variables con demasiados valores faltantes (>30%)
  • • Las variables con alta puntuación de relevancia suelen ser mejores predictores
  • • Incluye al menos 3-5 variables para un modelo robusto
  • • Evita variables que sean consecuencia del objetivo (data leakage)
