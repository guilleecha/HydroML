Preguntas Frecuentes

Una Fuente de Datos en HydroML es cualquier archivo que contiene datos que deseas analizar. Puede ser un archivo CSV, Excel (XLSX/XLS) o Parquet que contiene información sobre recursos hídricos, mediciones ambientales, o cualquier otro tipo de datos científicos.

Las fuentes de datos son la base de todos los análisis y experimentos en HydroML. Una vez que subes una fuente de datos a un workspace, puedes:

  • Visualizar y explorar los datos
  • Preparar y limpiar los datos
  • Crear experimentos de Machine Learning
  • Generar reportes de calidad de datos

HydroML acepta los siguientes formatos de archivo:

CSV
CSV

Archivos de valores separados por comas (.csv)

XLS
Excel

Hojas de cálculo (.xlsx, .xls)

PQ
Parquet

Formato de almacenamiento columnar (.parquet)

Todos los archivos se procesan y almacenan internamente en formato Parquet para optimizar el rendimiento y la compatibilidad.

Para crear un experimento de Machine Learning, sigue estos pasos:

  1. Prepara tus datos: Asegúrate de tener una fuente de datos subida a tu workspace
  2. Accede al workspace: Ve a la página de detalles de tu workspace
  3. Clic en "Nuevo Experimento ML": Encontrarás este botón en la esquina superior derecha
  4. Configura el experimento: Completa el formulario con:
    • Nombre descriptivo del experimento
    • Selecciona la fuente de datos
    • Elige la variable objetivo (target)
    • Selecciona el tipo de problema (regresión o clasificación)
    • Configura parámetros adicionales si es necesario
  5. Ejecuta el experimento: HydroML procesará automáticamente los datos y entrenará múltiples modelos

Consejo: Los experimentos se ejecutan en segundo plano. Recibirás notificaciones cuando el proceso inicie y termine.

Una Suite de Experimentos es una funcionalidad avanzada que te permite ejecutar múltiples experimentos de manera sistemática y comparar sus resultados.

Características principales:

  • Experimentos múltiples: Ejecuta varios experimentos con diferentes configuraciones
  • Comparación automática: Compara métricas de rendimiento entre experimentos
  • Optimización de hiperparámetros: Encuentra automáticamente los mejores parámetros
  • Reportes consolidados: Obtén un reporte completo con todos los resultados

Casos de uso típicos:

  • Comparar diferentes algoritmos en el mismo dataset
  • Probar múltiples configuraciones de preprocesamiento
  • Validar modelos con diferentes divisiones de datos
  • Optimizar parámetros de manera sistemática

Recomendación: Usa las Suites de Experimentos cuando necesites una comparación rigurosa entre múltiples enfoques de modelado.

HydroML incluye un potente conjunto de herramientas para preparar y limpiar tus datos antes del análisis:

Data Preparer:

  • Vista interactiva de datos con AG Grid
  • Cambio de tipos de datos de columnas
  • Eliminación de columnas no deseadas
  • Visualización de distribuciones con gráficos Plotly

Transformaciones Disponibles:

  • Imputación: Rellenar valores faltantes con media/mediana
  • Codificación: One-hot encoding para variables categóricas
  • Discretización: Convertir variables continuas en rangos

Fusión de Datos:

Combina múltiples fuentes de datos basándose en columnas comunes para crear datasets más ricos.

Los resultados de los experimentos incluyen métricas detalladas para evaluar el rendimiento de los modelos:

Para Regresión:

  • MAE: Error Absoluto Medio
  • MSE: Error Cuadrático Medio
  • RMSE: Raíz del Error Cuadrático Medio
  • R²: Coeficiente de Determinación

Para Clasificación:

  • Accuracy: Precisión general
  • Precision: Precisión por clase
  • Recall: Sensibilidad o cobertura
  • F1-Score: Media armónica de precisión y recall

Nota: Los experimentos también incluyen gráficos de rendimiento, matrices de confusión y análisis de importancia de características para una interpretación completa.
